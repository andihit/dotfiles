# My dotfiles

## Setup
```
git clone --recursive https://github.com/andreasgerstmayr/dotfiles.git .dotfiles
```

Take a look at the `Makefile` and install the things you want.

## Software
* zsh
* terminator

## WM
* i3-gaps, polybar, rofi, feh, dunst, compton
* sway, waybar, rofi, mako
